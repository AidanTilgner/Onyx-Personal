Welcome

This is the Onyx Interpretation Server. It is responsible for all all Natural Language Processing performed by the system. This portal allows you to modify the training data and therefore the model. It's effectively a GUI that lets you modify the model.

On the other pages, navigatable via the navigation bar above, you'll find the different modifications you can make, as well as explanations as to what they are. If you have questions, please ask them, as this feature is still in development. You should know what you're doing before using it.

Interactive Training

This feature is not meant to be used in development mode. The server will detect an update and retrain the model every time the training data updates. This causes a full server reload every time you make an update.

Here you can test the model with the text input below. Simply type something, press enter, and you'll get a response in the text box. You can then edit the expected intent for the text. Or if you think a different action should map to the intent given, you can edit that too. Finally, if you think a response doesn't fit, or another one should be added to an action, you can edit that too.

Tip: Check the other list views before changing actions or intents
